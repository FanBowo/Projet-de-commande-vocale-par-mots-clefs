Descritpion des codes:

- 'data_analysis' : est le code principal utilisé pour réaliser l'analyse du dataset qui 
			nous a été fourni.

- 'code-CNN-10labels-final' : est le code utilisé pour le modèle de notre système à base de
			couches CNN pour reconnaître 10 labels parmi les 35 existants.
			(Entrée : spectrogrammes)

- 'code-CNN-35labels-final' : est le code utilisé pour le modèle de notre système à base de 
			couches CNN pour reconnaître tous les 35 labels existants.
			(Entrée : spectrogrammes)

- 'code-CNN+MFCC' : est le code utilisé pour le modèle de notre système à base de 
			couches CNN en utilisant pour entrée des MFCCs au lieu des pectrogrammes
			pour les autres codes. (Entrée : MFCCs)

- 'code-CNN+RNN' : est le code utilisé pour le modèle de notre système à base de 
			couches CNN suivies par des couches RNN en utilisant pour entrée 
			(Entrée : Spectrogrammes)
